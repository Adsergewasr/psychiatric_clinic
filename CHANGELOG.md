## [1.1.0] - 2025-12-31

### Добавлено
- Полный CRUD интерфейс для пациентов
- Формы создания/редактирования с валидацией
- Поиск и фильтрация пациентов
- Экспорт данных (CSV, Excel, JSON)
- Дашборд со статистикой
- Красивые шаблоны с Bootstrap 5

### Исправлено
- Ошибки шаблонов с None значениями
- Проблемы с импортами и URL маршрутами
- Валидация данных в формах

### Изменено
- Улучшена админ-панель
- Добавлены дополнительные поля в модель Patient
- Оптимизированы запросы к базе данных

## [1.1.1] - 2026-01-01

### Исправлено
- Ошибка 'users is not a registered namespace' в шаблонах
- Временное использование admin:login и admin:logout для аутентификации
- Неправильные URL в навигации для неавторизованных пользователей

### Изменено
- Обновлены настройки редиректов после входа/выхода

## [2.0.0] - 2026-01-01

### Добавлено
- Кастомная модель User с расширенными полями
- Система ролей (Администратор, Врач, Медсестра, Регистратор, Аналитик)
- Модель UserProfile для дополнительных данных пользователя
- Модель LoginHistory для аудита входов
- Полный CRUD интерфейс для управления пользователями
- Кастомные формы аутентификации и регистрации
- Страницы профиля пользователя с историей активности
- API для получения списка врачей
- Интеграция ролевой системы с основным приложением

### Изменено
- Переход с django.contrib.auth на кастомную модель User
- Обновлены настройки аутентификации
- Изменены шаблоны для работы с новой системой пользователей
- Добавлены проверки прав доступа на основе ролей

### Исправлено
- Проблемы с аутентификацией и URL namespaces
- Ошибки шаблонов с None значениями
