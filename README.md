# Психиатрическая больница - Учет пациентов

Система учета пациентов психиатрической больницы по форме №003/у.

## Возможности

- Учет пациентов (CRUD операции)
- Админ-панель Django
- Импорт/экспорт данных (CSV, XLSX)
- Поиск и фильтрация пациентов
- Авторизация пользователей с ролями
- Выгрузка архивов за период
- Отправка карточек по email

## Технологии

- Python 3.13.9
- Django 6.0
- PostgreSQL
- Bootstrap 5
- Alpine.js
- django-import-export

## Установка

1. Клонировать репозиторий
2. Создать виртуальное окружение
3. Установить зависимости
4. Настроить базу данных
5. Применить миграции
6. Создать суперпользователя
